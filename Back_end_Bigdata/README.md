## :clipboard: 구현 기능
### 백엔드
* __회원__
   * 회원가입
   * 로그인/로그아웃
   * 구글 로그인(REST API) 
   * 사용자 권한관리( USER / STAFF / MASTER )
   * 사용자 아바타
<br>

* __게시판__
   * 게시글 리스트
   * 상세페이지
   * 게시글&댓글 CRUD
   * 이미지 & 파일 업로드 및 다운로드
   * 카테고리, 태그
   * 검색, 페이징
   * 카카오맵 (REST API)
   * KIBANA 대시보드 (API)
