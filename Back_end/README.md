## :clipboard: 구현 기능
### 백엔드
* __회원__
   * 회원가입
   * 로그인/로그아웃
   * 회원정보 수정 / 회원탈퇴
   * 카카오 로그인(REST API) 
<br>

* __고객센터__
   * 공지사항 게시판&댓글 CRUD
   * 문의사항 게시판&댓글 CRUD
<br>

* __장바구니(ssesion)__
   * 담기
   * 수량수정
   * 삭제
   * 쿠폰 적용
<br>

* __결제__
   * IAMPORT 결제기능 (Ajax)    
<br>

* __쿠폰__
   * 쿠폰 생성
<br>

* __상품__
   * 카테고리
   * 상품 리스트
   * 상품 등록
   * 상품 리뷰
